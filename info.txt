How do we deploy both frontend and backend together?
- The dist folder does the same task that the react application does.
- So we deploy the backend folder + dist folder on "Render"

Steps to deploy frontend and backend together:
- run "npm run build" in frontend folder, this will give you a dist folder that contains html, css and js files
- create a folder called public inside "Backend" folder and move the 3 files of dist folder inside public folder
- use middle ware app.use(express.static("./public")); //makes the files in public folder accessible to browser/user directly
- if i write 'src' instead of public in the middleware app.use(express.static("./src")); //this makes all the files in src folder accessible to the user directly including the app.js files
- whichever folder name u write in app.use(express.static(" ")) , the resources or assests of those folder are made publicly available
- By doing this, u can access the frontend folder on backend server and both are deployed together
